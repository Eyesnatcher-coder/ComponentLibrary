Avatar

The Avatar component is used to represent a user, displays the profile picture, initials or fallback icon.

Round Avatar

Easy UI has three circle shaped avatar which includes avatar large, avatar and avatar small. Size of Round Avatar is created by passing class="avatar round" with l or sm for large and small respectively. When nothing is passed it will load default as medium

Square Avatar

Easy UI has three square shaped avatar which includes avatar large, avatar and avatar small. Size of Sqaure Avatar is created by passing class="avatar" with l or sm for large and small respectively. When nothing is passed it will load default as medium
